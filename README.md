### 🚀 About Me

I am a highly motivated **DevOps & Cloud Engineer** with a strong focus on building and launching scalable **SaaS products**. I specialize in constructing resilient cloud environments, automating developer workflows, and implementing robust architectures that power modern applications.

- 🔭 **I’m currently working on:** Architecting high-availability infrastructure for cloud-native SaaS applications.
- 🌱 **I’m currently learning:** Advanced GitOps practices, service mesh implementation, and deep system profiling.
- 💬 **Ask me about:** Infrastructure as Code (IaC), CI/CD pipelines, Docker, Kubernetes, and cloud security practices.
- ⚡ **Fun fact:** I believe in automating everything — if I have to do a task manually more than twice, it gets a script!
